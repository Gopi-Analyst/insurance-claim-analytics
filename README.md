# Insurance Claim Analytics Project

## Project Overview
This project analyzes insurance claim data to identify claim trends, risk patterns, and settlement efficiency using SQL, Python, Excel, and Power BI.

## Dataset
- Synthetic insurance claims dataset
- 500 claim records
- Multiple claim types: Auto, Health, Home, Life, Travel

## Tools & Technologies
- Excel
- SQL
- Python (Pandas)
- Power BI
- GitHub

## Key Objectives
- Analyze claim amounts and approval rates
- Identify high-risk and fraud-flagged claims
- Measure claim settlement timelines
- Build interactive dashboards for insights
